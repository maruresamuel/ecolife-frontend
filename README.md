## 📌 About

EcoLife is a mobile e-commerce application that connects 
eco-conscious customers with vendors who sell sustainable 
and environmentally friendly products. The platform 
encourages green living by tracking the environmental 
impact of every purchase made.

---

## ✨ Features

- 🛒 **Order Management** — Place, track and manage orders seamlessly
- 🌱 **Sustainability Tracking** — Monitor the environmental impact of your purchases
- 🏪 **Vendor Dashboard** — Vendors can manage products, orders and earnings
- 💰 **Wallet & Payments** — Secure in-app wallet for fast and easy payments

---

## 🛠️ Tech Stack

### Frontend
- **Flutter** (Dart) — Cross-platform mobile development
- **Provider** — State management

### Backend
- **Node.js** — Server side logic and REST API

## 🚀 Getting Started

### Prerequisites
- Flutter SDK installed
- Node.js installed
- Android Studio or VS Code

### Frontend Setup
```bash
# Clone the repository
git clone https://github.com/maruresamuel/ecolife-frontend.git

# Navigate to the project
cd ecolife-frontend

# Install dependencies
flutter pub get

# Run the app
flutter run
```

### Backend Setup
```bash
# Clone the repository
git clone https://github.com/maruresamuel/ecolife-backend.git

# Navigate to the project
cd ecolife-backend

# Install dependencies
npm install

# Start the server
npm start
```
